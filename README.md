
## SNHU-CS210

A modular C++ console application for reading input data, calculating item frequencies, and writing processed results to an output file.

## Overview

This project was built in C++ as part of SNHU CS210. It demonstrates object-oriented design, file handling, and menu-driven console interaction by reading structured input data, analyzing frequency counts, and generating output results.

## Features

- Reads data from an input file
- Counts item frequencies
- Displays frequency information through a console menu
- Writes processed frequency data to an output file
- Organizes logic into separate manager classes for cleaner structure

## Technologies Used

- C++
- CMake
- File I/O
- Object-oriented programming

## Project Structure

The application is organized into modular components, including:

- `FileManager` for handling file input and output
- `FrequencyManager` for counting and managing frequency data
- `MenuManager` for user interaction and menu flow
- `main.cpp` as the application entry point

## Learning Goals

This project demonstrates:

- Class-based program design in C++
- File reading and writing
- Separation of concerns through modular structure
- Menu-driven console application design
- Basic data analysis using frequency counts

## How It Works

1. The program reads item data from an input file.
2. It calculates how often each item appears.
3. Users interact with the program through a console menu.
4. The results can be displayed in the console and written to an output file.

## Example Use Cases
